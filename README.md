# Inspectable Offline AI CLI System

## Problem
Most AI chatbots depend on cloud APIs and act as black boxes.

## Solution
A fully offline, explainable, CLI-based AI system with:
- Streaming responses
- Token usage tracking
- Executable tools
- Reasoning trace inspection
- Session audit & replay

## Architecture
- Modular engine
- Skill graph
- Tool execution layer
- Inspectable reasoning

## Commands
/skills  
/tool time  
/tool calc <expr>  
/explain  
/replay  

## Why This Is Different
This is not a chatbot demo.
This is an auditable AI system designed for reliability and future LLM integration.
